# 🏨 Hotel Booking Cancellations — Excel Dashboard
Excel dashboard for hotel booking cancellations analysis
## 📸 Dashboard Preview

![Dashboard](dashboard.png)

## 📌 Project Overview
This project analyzes hotel booking cancellations using a real-world dataset of **1,19,390 bookings** across City Hotel and Resort Hotel from 2015 to 2017.

The goal is to understand **why bookings get cancelled** and find patterns using Excel pivot tables and charts.
-------------------------------------------------
## 📊 Dataset Details

| Column | Description |
|---|---|
| hotel | City Hotel or Resort Hotel |
| is_canceled | 1 = Cancelled, 0 = Not Cancelled |
| arrival_date_year | Year of arrival (2015–2017) |
| arrival_date_month | Month of arrival |
| adults / children / babies | Number of guests |
| country | Guest's country |
| reserved_room_type | Room type reserved |
| assigned_room_type | Room type actually assigned |
| room_status | Desired or Un-Desired |
| Guest_type | Couple / Family / Single |

---
## 🔍 Key Findings

- 📌 **Overall cancellation rate is 37%** — 1 in every 3 bookings gets cancelled
- 📌 **City Hotel** has a higher cancellation rate (41.7%) compared to **Resort Hotel** (27.8%)
- 📌 **August** is the busiest month with 13,877 total bookings
- 📌 Guests who got **Un-Desired rooms** cancel far less (5.4%) vs Desired room guests (41.6%)
- 📌 **Couples** have the highest cancellation rate (39.8%) among all guest types
